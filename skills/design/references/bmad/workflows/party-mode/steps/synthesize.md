## Synthesis Guidelines

1. **Identify Patterns**
   - What did multiple agents emphasize?
   - Where do perspectives align?

2. **Acknowledge Tensions**
   - Don't hide disagreements
   - Provide balanced resolution paths

3. **Prioritize Actions**
   - Rank by impact and urgency
   - Consider resource constraints

4. **Maintain Traceability**
   - Link recommendations to agent insights
   - Show reasoning chain

## Quality Checklist
- [ ] All agent perspectives represented
- [ ] Conflicts addressed, not ignored
- [ ] Recommendations are actionable
- [ ] Next steps are concrete
