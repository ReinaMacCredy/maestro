# Prototype

## Purpose
Build quick, low-fidelity representations to test ideas.

## Activities
- Paper prototyping
- Wireframing
- Storyboarding
- Role playing
- Wizard of Oz testing

## Principles
- Start rough, iterate
- Fail fast, learn fast
- Build to think
- Keep it simple

## Outputs
- Testable prototypes
- Interaction flows
- Key assumptions identified
- Test plan
