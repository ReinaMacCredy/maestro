# Define

## Purpose
Synthesize research into actionable problem statements.

## Activities
- Affinity mapping
- Problem framing
- POV statement creation
- How Might We questions

## Outputs
- Problem statement
- Point of View (POV)
- HMW questions
- Design challenge

## Template
**POV Statement:**
[User] needs [need] because [insight].

**HMW Format:**
How might we [action] for [user] so that [outcome]?
