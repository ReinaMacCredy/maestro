# Structure

Choose and apply a story structure.

## Objective
Select the narrative framework that best fits the message.

## Structures
- **Hero's Journey**: Transformation through challenge
- **Problem-Solution**: Pain → Relief
- **Before-After**: State change visualization
- **Three Act**: Setup → Conflict → Resolution

## Actions
1. Match structure to audience and goal
2. Define key story beats
3. Place the product as guide/tool
4. Build tension and release

## Output
- Chosen structure
- Story outline
- Key moments
