# Frozen Google Style Guide Snapshots

These files are frozen snapshots used by `maestro-setup`. Setup must copy only
the guides detected for the current repository into
`.maestro/context/code_styleguides/`.

Source index: https://google.github.io/styleguide/

The Google style guides in this directory are primarily licensed under CC-BY
3.0. Some copied pages separately license code samples under Apache 2.0. Keep
the source URL, snapshot date, attribution, and per-file license note at the top
of each copied guide.

V1 intentionally excludes external Dart and Kotlin guides because they live
outside the Google styleguide project and need separate licensing approval.
