# Tell

Craft and deliver the narrative.

## Objective
Write the story with emotional resonance and clarity.

## Actions
1. Open with a hook
2. Build empathy for the hero
3. Show the transformation
4. Use concrete details and sensory language
5. End with clear call to action

## Principles
- Show, don't tell
- Make it personal
- Keep it simple
- Create emotional connection

## Output
- Complete narrative
- Supporting visuals/assets
- Delivery format
