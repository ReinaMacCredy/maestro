# Test

## Purpose
Validate solutions with real users and gather feedback.

## Activities
- Usability testing
- A/B testing
- Feedback sessions
- Observation
- Iteration planning

## Questions
- Does this solve the problem?
- Is it usable?
- Is it desirable?
- What needs improvement?

## Outputs
- User feedback
- Validated learnings
- Iteration priorities
- Next steps

## Iteration
Return to earlier phases as needed based on learnings.
